# PluginState

| 名称                    | 说明       |
|-----------------------|----------|
| DISABLED              | 被禁言      |
| VERSION_INCOMPATIBLE  | 版本不兼容    |
| PLATFORM_INCOMPATIBLE | 平台不兼容    |
| INACTIVATED           | 插件未激活    |
| DEPENDS_NOT_RUNNING   | 依赖插件没有运行 |
| STOPPED               | 暂停       |
| RUNNING               | 运行中      |
